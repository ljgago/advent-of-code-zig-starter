## Usage

The project is configured for `run`, `test` and `install` each day independently.
You can see the available commands with:

    $ zig build -h

For example:

    # only run the day01
    $ zig build day01-run

    # only test the day02
    $ zig build day02-test

    # only install the day03
    $ zing build day03-install

Folder structure:

    src
    └── day01
        ├── day01.zig
        ├── input.txt
        └── README.md
